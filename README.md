# Kopilotti — Myyjän co-pilot

Reaaliaikainen tekoälyassistentti autokaupan myyjille. Kopilotti kuuntelee asiakaskohtaamisen, tunnistaa ostosignaalit ja tarjoaa myyjälle tilannekohtaisia vihjeitä — kesken tapaamisen, heti kun niitä tarvitaan.

**Demo:** https://mikko-lab.github.io/kopilotti-demo/

## Mitä se tekee

- Tunnistaa puheesta hintaepäilyt, rahoituskiinnostuksen, vaihtoauton ja perheen tarpeet
- Näyttää ostohalukkuusmittarin (0–100 %) reaaliajassa
- Ehdottaa sopivia autoja varastosta tunnistettujen signaalien perusteella
- Generoi CRM-yhteenvedon kohtaamisesta yhdellä klikkauksella

## Kokeileminen

Avaa demo selaimessa. Klikkaa demo-skenaarioita (Hintaepäily, Rahoitus, Perhe jne.) tai aloita oikea sessio mikrofonilla.
